Violy Documentation

Thank You for purchasing this template, I hope You enjoy it!

1. Template Structure

  • - You can find all files in template folder
  • - In template folder you will see build and dev folders
  • - build folder contain all compiled and ready to use files
  • - dev folder contain all sources for your work
  • - CSS, js, images, fonts files you can find in assets folder
  • - CSS files you can find in build/assets/css folder or dev/assets/sass folder with sources
  • - JS files you can find in build/assets/js folder or dev/assets/js folder with sources
  • - Fonts you can find in assets/webfonts folder

2. How to edit template (for users)

Better way to use sources in dev folder, but if you don't know all technologies you can simply edit build files

  • - Change intro background image. Upload your image to assets/img/ folder. Open index.html file and find style="background-image: url('http://placehold.it/1980x700/333');" or img tag. Change url value to your image path. F.e. url("../img/image.jpg")
  • - Change other images. Upload your image to assets/img/ folder. Open any style.css file and find url('http://placehold.it/1980x700/333'). Change the url to your image path.
  • - All texts you can change in .html files

3. How to edit template (for developers)

If you want to edit some html, css, js files, best way to use sources in dev folder. You need node.js, gulp installed

  • 1. Open cmd and go to the projects folder
  • 2. Run npm i to install all gulp dependencies
  • 4. Run gulp to clean build folder, build the new project, run local server and start watching changes
  • 4.2 Run gulp build to build project
  • 4.3 Run gulp watch to watch changes
  • 4.4 Run gulp build to build only html files
  • 4.5 Run gulp sass to build only css files
  • 4.6 Run gulp fonts to build only fonts
  • 4.7 Run gulp img to build only images
  • 4.8 Run gulp clean to delete build folder

How to work with html

  • - To change html find dev folder which includes all .pug files.
  • - You can see templates folder which includes some blocks which are used in whole template (f.e. head, end)
  • - This blocks are included in all pages with //= includes/head.pug construction

How to change css

  • - To change css open dev/assets/sass folder
  • - main.scss - main file which import all blocks
  • - All files you can find in includes folder

How to change js

  • - To change js open dev/assets/js folder
